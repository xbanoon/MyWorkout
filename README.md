# تماريني — المرحلة الأولى 🏋️

تطبيق ويب لمتابعة جدول التمارين الأسبوعي (Workout A / Workout B) مع لوحة بيانات للتقدم والالتزام.

## المميزات

- 📅 جدول أسبوعي تفاعلي قابل للتعديل
- 🏋️ تتبع كل مجموعة على حدة (وزن + تكرارات)
- 📊 مقارنة بآخر أداء مسجل
- ⏱️ مؤقت راحة بين المجموعات مع تنبيه صوتي
- 📈 رسم بياني لتقدم الأوزان
- 🌙 وضع ليلي
- 📱 PWA — قابل للتثبيت ويعمل أوفلاين
- 💾 تصدير / استيراد البيانات

## التشغيل

افتح `index.html` مباشرة، أو زر النسخة المنشورة:
**https://xbanoon.github.io/MyWorkout/**

كل البيانات تُحفظ محلياً في المتصفح (localStorage).
